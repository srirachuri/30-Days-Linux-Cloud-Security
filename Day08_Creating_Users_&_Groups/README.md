# Day 8 – Creating Users & Groups

## Goal
Learn how to create and manage users and groups in Linux, which is essential for access control and security.

## Commands Practiced
- useradd
- passwd
- groupadd
- usermod
- id
- su
- whoami

## What I Did
- Created new users with home directories
- Set passwords for users
- Created groups
- Added users to groups
- Verified user and group details
- Switched users and confirmed identity

## Security Relevance
User and group management is the foundation of:
- Linux permissions
- Role-based access control (RBAC)
- Cloud IAM concepts

## Goal
Understand and control file and directory permissions using chmod, chown, and chgrp for secure Linux administration.

## Concepts Learned
- Read, write, execute permissions
- Numeric and symbolic permission modes
- File ownership and group ownership
- Sticky bit usage
- Real-world permission best practices

## Commands Practiced
- ls -l
- chmod
- chown
- chgrp

## What I Did
- Created files and inspected default permissions
- Applied secure permissions for private and shared files
- Changed file ownership and group ownership
- Verified permissions using ls -l
- Completed a mini security challenge

## Security Relevance
File permissions are critical for:
- Preventing unauthorized access
- Protecting sensitive data
- Securing web servers and SSH access
- Enforcing least privilege

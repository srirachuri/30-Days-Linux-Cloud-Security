# Day 9 – Managing Users in Linux

## Goal
Learn how to modify, lock, unlock, and delete users and groups in Linux for secure system administration.

## Commands Practiced
- usermod
- userdel
- groupdel
- passwd
- su
- id

## What I Did
- Created and modified user accounts
- Renamed users and updated home directories
- Added users to groups
- Locked and unlocked user accounts
- Deleted users and groups securely
- Verified changes using id and whoami

## Security Relevance
User management is critical for:
- Access control
- Incident response
- Audit and compliance
- Cloud IAM lifecycle management
